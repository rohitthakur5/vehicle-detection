# 🚗 Vehicle Detection & Counting System

## 📌 Overview

This project detects and counts vehicles in a video using OpenCV.
Vehicles are counted when they cross a predefined line.

---

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy

---

## ▶️ How to Run

1. Install dependencies:

```bash
pip install opencv-python numpy
```

2. Run the code:

```bash
python main.py
```

---

## ⚙️ Working

* Detects moving objects using background subtraction
* Filters noise using image processing
* Counts vehicles when they cross a line

---

## 📸 Output

* Vehicles are detected with bounding boxes
* Total count is displayed on screen

---

## ⚠️ Limitation

* Same vehicle may be counted multiple times

---
