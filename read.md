
Vehicle Counting System
📌 Overview

This project detects and counts vehicles in a video using OpenCV.
Vehicles are counted when they cross a predefined line.

🛠️ Tech Stack
Python
OpenCV
NumPy
▶️ Run
pip install opencv-python numpy
python main.py
⚙️ How It Works
Detect moving objects using background subtraction
Apply filters to remove noise
Count vehicles when they cross a line
